# Pokémon Evolution Viewer

¡Bienvenidos a Pokémon Evolution Viewer!  
Esta es una aplicación sencilla hecha con **React** para mostrar las evoluciones de los Pokémon utilizando la **PokéAPI**. Además, he utilizado **Sass** para gestionar los estilos de manera más eficiente.

## ¿Qué hace esta aplicación?
La app te permite:
- Buscar un Pokémon por su nombre o número de Pokédex.
- Ver su cadena evolutiva completa.
- Explorar detalles adicionales de cada evolución, como el tipo o estadísticas básicas.

## Tecnologías usadas
- **React**: Para la estructura y la lógica de la aplicación.
- **Sass**: Para manejar los estilos de manera más organizada y modular.
- **PokéAPI**: Como fuente de datos para obtener la información de los Pokémon y sus evoluciones.

## ¿Cómo ejecutarlo en tu máquina?
1. Clona este repositorio:

   ```bash
   git clone https://github.com/tu-usuario/nombre-del-repo.git
   ```

2. Instala las dependencias

   ```bash
   npm install
   ```

3. Inicia la aplicación

   ```
   npm start
   ```
